This release of Hazelcast introduces the migration tool to be used between IMDG 3.12 and Platform 5.3.1 versions.
For this, we've added the support for processing WAN replication messages from Hazelcast IMDG 3.x cluster members (see https://github.com/hazelcast/hazelcast/pull/24910).

See also
* https://docs.hazelcast.com/hazelcast/latest/migrate/migration-tool-imdg to learn how to use the migration tool.
* https://docs.hazelcast.com/hazelcast/latest/migrate/upgrading-from-imdg-3 to learn the technical changes between the two versions.
